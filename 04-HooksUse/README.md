# React Hooks Practice

A React.js practice project focused on understanding and implementing React Hooks. This project demonstrates how hooks can be used to manage state and add functionality to functional components without using class components.

## 🚀 Features

* Learn and practice React Hooks
* State management using `useState`
* Dynamic UI updates
* Functional component architecture
* Clean and reusable code structure

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/rewati1616/React.js-Practice-code.git
```

2. Navigate to the project folder:

```bash
cd React.js-Practice-code/04-HooksUse
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```text
http://localhost:5173
```
