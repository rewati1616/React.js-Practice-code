# 💼 JobHub - Job Portal UI

JobHub is a modern and responsive job portal web application built with React.js. It allows users to browse job opportunities, explore companies, and navigate through a clean and user-friendly interface.

## 🚀 Features

* Modern and Responsive UI
* Job Listings with Detailed Information
* Company Profiles
* Search Jobs Interface
* Login Page
* Sign Up Page
* Interactive Navigation Bar
* Job Categories Dropdown
* Companies Dropdown
* Professional Footer Section
* Reusable React Components

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router DOM
* Lucide React Icons
* Vite

## 📋 Job Information Displayed

Each job card includes:

* Company Logo
* Company Name
* Job Position
* Employment Type
* Experience Level
* Salary Information
* Location
* Posted Date
* Apply Button
* Save Job Feature

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/rewati1616/React.js-Practice-code.git
```

### Navigate to Project Folder

```bash
cd React.js-Practice-code/01-FristProject
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
